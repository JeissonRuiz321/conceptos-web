Conceptos clave de desarrollo web

El desarrollo web implica tareas como diseño visual, usabilidad, codificación, optimización móvil y gestión de bases de datos.

  • HTML
  • CSS
  • JavaScript
  • Git
  • GitHub

Incluye la creación de interfaces interactivas, la optimización para dispositivos móviles y la implementación de funcionalidades específicas.

Arquitectura Cliente-Servidor

Modelo que divide tareas entre servidores (proveedores) y clientes (solicitantes).

  • Servidor web
  • Bases de datos
  • Lenguajes de programación
  • Framework
  • Librerías

Permite separar la lógica del cliente y del servidor, facilitando el mantenimiento.

Diseño centrado en el usuario

El usuario es el eje del diseño web, priorizando sus necesidades sobre las capacidades técnicas.

  • Diseño de interfaz
  • Diseño de experiencia
  • Diseño visual

Mejora la usabilidad y la satisfacción del usuario.

Investigación en el entorno

Identifica factores internos y externos que afectan el éxito de un proyecto.

  • Análisis de requisitos
  • Diseño de prototipo
  • Evaluación de solución
